### 介绍

### 开发

#### 安装依赖
> npm install
#### 调试
> npm run serve
#### 打包dist
> npm run build
#### 打包安卓安装包
此app用到DCloud的5+App技术，所以必须使用HBuilderX App开发版进行打包。
